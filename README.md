# Welcome!
Welcome to... **Word Guess**, where we will help you *Guess the Word!*

# Introductions
Follow the code in the _`guess_word.py`_ file above to begin!

# Instructions
Upon running the program, you will be automatically prompted to enter the length of the word(s) you are trying to guess. Then, the program will ask you if the word(s) contains any space(s), and if so, it will ask you how many spaces, and what the length of the ***FIRST*** word is, to help narrow down your options. If not, then you can skip the past steps and directly proceed to being asked if you know any letters in the word, and if so, you will be continuously prompted to enter the letter and which position in the word(s) it is in, which at any point you can choose to stop entering in more letters if you do not have any more. If you do not know any letters in the word ahead of time, you can skip the past steps, at which point the program will display, to the best of its ability, a list of all the possible words that you may be trying to guess.

If you get any additional information about the word later on, feel free to continue running the program, now taking into consideration the new information you have when entering in the new, additional details about the word(s) into the program.
